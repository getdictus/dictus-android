Create `app/src/main/java/dev/pivisolutions/dictus/service/AudioCaptureManager.kt`:
- Class wrapping AudioRecord API at 16kHz mono Float32 (ENCODING_PCM_FLOAT)
- Constructor takes no args; AudioRecord created in `start()`, released in `stop()`
- `start()`: creates AudioRecord with MediaRecorder.AudioSource.MIC, 16000 Hz, CHANNEL_IN_MONO, ENCODING_PCM_FLOAT, bufferSize = AudioRecord.getMinBufferSize(...) * 2. Calls recorder.startRecording(). Launches coroutine on provided CoroutineScope (Dispatchers.Default) to read samples in a loop.
- Read loop: reads into temporary FloatArray, appends to main `samples: ArrayList<Float>` (synchronized), calculates RMS energy, calls `onEnergyUpdate: (Float) -> Unit` callback with normalized value.
- `stop()`: cancels read job, calls recorder.stop(), recorder.release(). Returns collected samples as FloatArray.
- `cancel()`: cancels read job, stops and releases recorder, clears samples.
- `calculateRmsEnergy(buffer: FloatArray, count: Int): Float` -- public for testability. Returns 0f if count==0, else sqrt(sum of squares / count).
- `normalizeEnergy(rms: Float): Float` -- returns (rms * 5f).coerceIn(0f, 1f).
- Energy history: internal ArrayDeque<Float>(30), addLast + removeFirst if size > 30. getEnergyHistory() returns toList().

Key implementation details:
- Inner class `LocalBinder : Binder()` with `fun getService(): DictationService`
- `_state = MutableStateFlow<DictationState>(DictationState.Idle)` / `val state: StateFlow<DictationState> = _state.asStateFlow()`
- `serviceScope = CoroutineScope(SupervisorJob() + Dispatchers.Main)` -- cancelled in onDestroy()
- Timer job: launched in startRecording(), increments elapsedMs every 1000ms, updates _state with current energy from AudioCaptureManager
- Companion object constants: `CHANNEL_ID = "dictus_recording"`, `NOTIFICATION_ID = 1`, `ACTION_START = "dev.pivisolutions.dictus.action.START"`, `ACTION_STOP = "dev.pivisolutions.dictus.action.STOP"`

`onBind(intent)`: return LocalBinder instance

`onStartCommand(intent, flags, startId)`:
- When ACTION_START: call createNotificationChannel(), then ServiceCompat.startForeground(this, NOTIFICATION_ID, buildNotification(), ServiceInfo.FOREGROUND_SERVICE_TYPE_MICROPHONE), then startAudioCapture(). Return START_NOT_STICKY.
- When ACTION_STOP: call stopRecordingInternal(discard=true), stopForeground(STOP_FOREGROUND_REMOVE), stopSelf(). Return START_NOT_STICKY.

`createNotificationChannel()`: create NotificationChannel(CHANNEL_ID, "Dictus Recording", NotificationManager.IMPORTANCE_LOW) with setShowBadge(false), description "Active recording notification".

`buildNotification()`: NotificationCompat.Builder with contentTitle "Dictus - Recording", smallIcon R.drawable.ic_mic, setOngoing(true), addAction(R.drawable.ic_stop, "Stop", stopPendingIntent). The stopPendingIntent is PendingIntent.getService with ACTION_STOP intent, FLAG_UPDATE_CURRENT or FLAG_IMMUTABLE. setContentIntent to null (no-op on body tap per user decision).

`fun startRecording()`: dispatches ACTION_START via startForegroundService (for external callers like the IME).

`fun stopRecording(): FloatArray`: stops capture via audioCaptureManager.stop(), cancels timer, updates _state to Idle, stops foreground + self. Returns the float buffer.

`fun cancelRecording()`: calls audioCaptureManager.cancel(), cancels timer, updates _state to Idle, stops foreground + self.

`startAudioCapture()`: creates AudioCaptureManager, calls start(serviceScope) with onEnergyUpdate callback that updates _state to Recording(elapsedMs, energyHistory). Starts timer coroutine.

`onDestroy()`: serviceScope.cancel(), Timber.d("DictationService destroyed").
