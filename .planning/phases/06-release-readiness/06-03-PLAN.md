3. **Add uiLanguage to SettingsViewModel** (GREEN phase):
   - Add a `val uiLanguage: StateFlow<String>` that reads from `AppCompatDelegate.getApplicationLocales()`:
     ```kotlin
     val uiLanguage: StateFlow<String> = MutableStateFlow(getCurrentUiLanguage())

     private fun getCurrentUiLanguage(): String {
         val locales = AppCompatDelegate.getApplicationLocales()
         return locales.toLanguageTags().ifEmpty { "system" }
     }

     fun setUiLanguage(languageTag: String) {
         val locales = if (languageTag == "system") {
             LocaleListCompat.getEmptyLocaleList()
         } else {
             LocaleListCompat.forLanguageTags(languageTag)
         }
         AppCompatDelegate.setApplicationLocales(locales)
         // Activity will recreate, so StateFlow update is not needed
     }
     ```
   - Add imports: `import androidx.appcompat.app.AppCompatDelegate` and `import androidx.core.os.LocaleListCompat`
   Run tests — they should now PASS (GREEN).
