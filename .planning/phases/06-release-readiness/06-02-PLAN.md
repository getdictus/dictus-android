  <action>
1. **Add appcompat dependency to version catalog:**
   In `gradle/libs.versions.toml`:
   - In [versions] section, add: `appcompat = "1.7.0"`
   - In [libraries] section, add: `androidx-appcompat = { group = "androidx.appcompat", name = "appcompat", version.ref = "appcompat" }`

2. **Add appcompat to app/build.gradle.kts:**
   In the dependencies block, add: `implementation(libs.androidx.appcompat)`

3. **Enable generateLocaleConfig in app/build.gradle.kts:**
   Inside the `android {}` block, add:
   ```kotlin
   androidResources {
       generateLocaleConfig = true
   }
   ```

4. **Create app/src/main/resources.properties:**
   ```properties
   unqualifiedResLocale=en-US
   ```
   This tells AGP that values/strings.xml is English, so the auto-generated LocaleConfig correctly declares `en` and `fr` as supported locales.

5. **Add AppLocalesMetadataHolderService to AndroidManifest.xml:**
   Inside the `<application>` tag in `app/src/main/AndroidManifest.xml`, add:
   ```xml
   <service
       android:name="androidx.appcompat.app.AppLocalesMetadataHolderService"
       android:enabled="false"
       android:exported="false">
       <meta-data
           android:name="autoStoreLocales"
           android:value="true" />
   </service>
   ```

6. **Change MainActivity to extend AppCompatActivity:**
   In `app/src/main/java/dev/pivisolutions/dictus/MainActivity.kt`:
   - Replace `import androidx.activity.ComponentActivity` with `import androidx.appcompat.app.AppCompatActivity`
   - Change `class MainActivity : ComponentActivity()` to `class MainActivity : AppCompatActivity()`
   - Keep all other code unchanged. AppCompatActivity extends FragmentActivity extends ComponentActivity, so all existing APIs (setContent, enableEdgeToEdge, etc.) remain available.
   - Note: `enableEdgeToEdge()` is from `androidx.activity.enableEdgeToEdge`, which works on AppCompatActivity since it extends ComponentActivity. Verify this import is retained.
