**Part D: Adapt ModelDownloader for tar.bz2 archives**

Parakeet downloads are `.tar.bz2` archives that must be extracted. Modify the download logic:

1. In `ensureModelAvailable()`, after downloading, if the model is a Parakeet model, extract the archive. The archive typically contains a directory like `sherpa-onnx-nemo-parakeet_tdt_ctc_110m-en-36000-int8/` with `model.int8.onnx` and `tokens.txt` inside.

2. Add extraction logic using `java.util.zip` or `org.apache.commons.compress`. Since the project doesn't have commons-compress, use a simpler approach:
   - Download the tar.bz2 to a temp file
   - Use `ProcessBuilder` to run `tar xjf` (available on Android) to extract
   - Move the extracted model.onnx and tokens.txt to `models/{key}/`
   - Delete the temp archive

3. Alternative simpler approach: Use `Runtime.getRuntime().exec("tar xjf ...")` for extraction. If `tar` is not available on the device, fall back to a Java-based BZip2 + Tar extraction using `java.io` streams.

4. The `downloadWithProgress()` flow should also handle archive extraction for Parakeet models. After the download completes, extract before emitting `Complete`.

5. Key extraction logic:
```kotlin
private fun extractTarBz2(archivePath: String, targetDir: String, modelInfo: ModelInfo): Boolean {
    val targetDirFile = File(targetDir)
    targetDirFile.mkdirs()

    // Use tar command available on Android (via toybox)
    val process = ProcessBuilder("tar", "xjf", archivePath, "-C", targetDirFile.parent)
        .redirectErrorStream(true)
        .start()
    val exitCode = process.waitFor()
    if (exitCode != 0) {
        Timber.e("tar extraction failed with exit code %d", exitCode)
        return false
    }

    // The archive extracts to a directory like sherpa-onnx-nemo-parakeet...
    // Find the extracted directory and move model files to targetDir
    val extractedDirs = targetDirFile.parentFile?.listFiles { f ->
        f.isDirectory && f.name.startsWith("sherpa-onnx-nemo")
    }
    val extractedDir = extractedDirs?.firstOrNull() ?: return false

    // Move model file and tokens.txt
    File(extractedDir, modelInfo.fileName).renameTo(File(targetDir, modelInfo.fileName))
    File(extractedDir, "tokens.txt").renameTo(File(targetDir, "tokens.txt"))

    // Cleanup extracted directory
    extractedDir.deleteRecursively()
    return true
}
```
